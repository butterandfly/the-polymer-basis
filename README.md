# the-polymer-basis
Learn the basis of polymer now.

## Step 0
安装以后需要的npm及bower依赖：

```
# 确保已安装node及npm

# 安装npm依赖
npm install
```

```
# 确保已安装bower

# 安装bower依赖
bower install
```

```
# 安装http-server，一个基于node的简易服务器
npm install -g http-server
```

## 开始学习
完成“step 0”里面的所有准备工作后，可以开始我们的教程了。
